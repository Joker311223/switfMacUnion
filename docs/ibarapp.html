核心功能
专为刘海屏设计
MacBook Pro 的刘海会遮住菜单栏中间区域，iBarApp 用覆盖层把刘海旁的图标收起，还给左侧菜单更多空间。
🍎
解决刘海遮挡问题
MacBook Pro 14/16 寸的刘海会遮住菜单栏中间的图标。iBarApp 把刘海右侧紧邻的可见图标收入覆盖层，腾出空间给左侧应用菜单。
◀▶
左键一键收起 / 展开
点击 ▶ 按钮，覆盖层从右向左以动画展开，把刘海旁图标收起；点击 ◀ 再次展开，图标重新可见，切换流畅自然。
📏
自定义收起宽度
偏好设置中调整覆盖层宽度（50–500 pt），每约 24pt 覆盖一个图标，精确控制收起范围，普通屏同样适用。
⏱
鼠标离开自动收起
鼠标离开覆盖区域后自动收起，延迟时间 1–10 秒可配置。展开查看图标后，放开鼠标即自动归位。
🌙
深色 / 浅色自适应
覆盖层颜色精确匹配 macOS 菜单栏背景，跟随系统外观自动切换，视觉上几乎无缝融入菜单栏。
🫥
零存在感后台运行
纯状态栏应用，无 Dock 图标，无主窗口。右键 ▶ 按钮可随时打开偏好设置或退出，极简无干扰。
使用方式
四步搞定
启动即用，右键按钮随时调整。适配刘海屏 MacBook Pro，普通屏同样可用。
1
构建并运行
使用 swift build 后运行，iBarApp 出现在菜单栏右侧，显示一枚 ▶ 按钮。
2
左键点击收起图标
单击 ▶，覆盖层从右向左动画展开，把刘海旁的图标遮住收起。再点 ◀ 展开，图标重新可见。
3
右键打开偏好设置
右键点击 ▶/◀ 按钮 → 「偏好设置」，拖动「收起宽度」滑块，约 24pt 覆盖一个图标，调到刚好盖住刘海旁的图标为止。
4
开启自动收起（可选）
勾选「鼠标离开后自动收起」，设置延迟（1–10 秒），展开查看图标后放开鼠标，覆盖层会自动归位。
偏好设置
细节可控
所有设置通过 UserDefaults 持久化，重启后自动恢复，让 iBarApp 始终按你的习惯工作。
📏 收起区域
  • 收起宽度调节（50–500 pt）
  • 约每 24pt 覆盖一个菜单栏图标
  • 拖动滑块实时调整效果
⏱ 自动行为
  • 启动时自动收起
  • 鼠标离开覆盖区域后自动收起
  • 自动收起延迟时间（1–10 秒）
🎨 外观
  • 覆盖层右侧分隔线显示 / 隐藏
  • 覆盖层颜色：跟随系统（推荐）/ 强制浅色 / 强制深色
构建与运行
一行命令，即刻运行
使用 Swift Package Manager 构建，无需 Xcode，无需额外依赖。
bash 
# 克隆仓库
git clone https://github.com/joker311223/switfMacUnion.git
cd switfMacUnion/iBarApp

# 构建（Debug）
swift build

# 运行（纯状态栏应用，无主窗口）
.build/debug/iBarApp

# 构建 Release 版本（推荐）
swift build -c release
.build/release/iBarApp

⚠️ 要求 macOS 13+ 及 Swift 5.9+（Xcode 15 或 Swift 命令行工具）
ℹ️ 启动后应用出现在菜单栏右侧，无 Dock 图标，无主窗口。
