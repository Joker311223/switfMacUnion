核心功能
专为写作而生
每一项功能都经过精心设计，消除写作过程中的每一个摩擦点。
⚡
Markdown 实时预览
编辑区与预览区并排布局，输入即渲染。支持标题、粗斜体、代码块、表格、高亮、引用等完整 Markdown 语法。
⌨
全局快捷键唤醒
在任意界面按 ⌃⌥Z，瞬间唤出最近编辑的备忘录。灵感来了，不必切换 App，立即记录。
🖼
截图直接粘贴
截图后 ⌘V 直接粘贴，自动保存为本地图片文件并在光标处插入 Markdown 图片引用。
🔗
超链接快速插入
选中文字后按 ⌘K，弹出输入框填入 URL，自动生成标准 Markdown 链接格式。
💾
自动 + 手动保存
停止输入 0.5 秒后自动保存，也可随时 ⌘S 手动保存。工具栏绿点 / 红点实时指示保存状态。
📌
窗口置顶
一键将 MemoApp 窗口钉住在所有窗口最前方，对照参考资料写作时再也不用频繁切换窗口。
🎨
完全可定制外观
自定义各级标题、正文、链接、代码块的颜色与字体，深色 / 浅色 / 跟随系统三种模式自由切换。
📤
批量导出
一键将全部备忘录导出为标准 Markdown 文件，数据随时可携带，不被任何平台锁定。
快捷键
双手不离键盘
完善的键盘快捷键覆盖所有常用操作，熟练后效率大幅提升。
快捷键功能描述适用范围
⌃⌥Z 全局唤醒 MemoApp 全局生效
⌘N 新建备忘录 应用内
⌘S 手动保存当前备忘录 应用内
⌘K 选中文字后插入超链接 编辑区
⌘F 全文查找 编辑区
⌘Z 撤销 编辑区
⇧⌘Z 重做 编辑区
⌘W 关闭窗口 应用内
偏好设置
按你的方式工作
六大设置面板，从启动行为到存储路径，每一处都可以调整到最适合你的状态。
⚙️ 通用
  • 启动时自动打开最近备忘录
  • 删除前弹出确认对话框
  • 显示字数与行数统计
✏️ 编辑器
  • 字体风格与字号调节
  • 行间距与段落间距
  • 自动换行开关
  • Markdown 语法高亮
  • 自动保存延迟时间（默认 0.5s）
🎨 外观
  • 深色 / 浅色 / 跟随系统
  • 预览区字号
  • 各级标题颜色自定义
  • 正文、链接、代码颜色
⌨ 快捷键
  • 全局快捷键速查表
  • 编辑器快捷键速查表
📁 存储
  • 自定义备忘录存储目录
  • 一键恢复默认存储路径
📤 导出
  • 导出格式选择（.md / .txt）
  • 单条 / 全部批量导出
构建与运行
一行命令，即刻运行
使用 Swift Package Manager 构建，无需 Xcode，无需额外依赖。
bash 
# 克隆仓库
git clone https://github.com/joker311223/switfMacUnion.git
cd switfMacUnion/MemoApp

# 构建（Debug）
swift build

# 运行
.build/debug/MemoApp

# 构建 Release 版本（推荐）
swift build -c release
.build/release/MemoApp

⚠️ 要求 macOS 13+ 及 Swift 5.9+（Xcode 15 或 Swift 命令行工具）
